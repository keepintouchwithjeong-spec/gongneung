🕒 중국어로 시간 말하는 방법


*참고: H=Hour(시), M=Minute(분)

1. 시각(시점)을 표현하는 기본 형태
H点M分
*10분 단위(10, 20, 30, 40, 50)에서는 分 생략하기도 함
*초는 秒miǎo

2. 2시 일 때
两点M分
*2시는 二이 아닌 两을 이용함에 유의

3. 정각일 때
H点(整)
*일반적으로 H点로만 말하나, 뒤에 정각을 강조한다면 "整"을 붙이기도 함

4. 30분일 때
H点三十分 또는 H点半
*半을 쓸때는 分 붙이지 않음

5. 15분일 때
H点十五分 또는 H点一刻
*一刻=15분을 의미, 刻를 쓸때는 半처럼 分 붙이지 않음

6. 45분일 때
H点四十五分 또는 H点三刻
*一刻=15분이기에 三刻는 45분을 의미, 刻를 쓸때는 半처럼 分 붙이지 않음

7. H시 1~9분일 때
H点(零)M分
*1~9분일 때는 零을 붙여주는 것이 일반적이나 생략하기도 함

8. '몇 시 몇 분 전'으로 표현할 때
差60-M分H+1点
*点과 分 위치가 달라짐에 유의
*15분 전은 十五分 대신 一刻 이용할 수도 있음
→ 差十五分H点 또는 差一刻H点

9. 시각 표현 예시
8:45 → 八点四十五分 / 八点三刻 / 差十五分九点 / 差一刻九点
8:15 → 八点十五分 / 八点一刻
8:00 → 八点 / 八点整
8:30 → 八点三十分 또는 八点半
8:05 → 八点零五分 / 八点五分



10. 기타 : 시간의 양 말하기
시간의 양(몇 시간 몇 분)을 말할때는 시간에 '个小时', 분에 分钟을 붙임
H个小时M分钟

1시간 → 一个小时
2시간 → 两个小时
30분 → 半个小时 / 三十分钟
15분 → 十五分钟 / 一刻钟
